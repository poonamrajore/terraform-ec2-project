# Terraform EC2 Web Server Project

## Project Overview
This project creates an EC2 instance using Terraform with:
- Default VPC (data block)
- Security Group allowing SSH and HTTP
- EC2 instance (t3.micro)
- Public IP output

## Technologies Used
- Terraform
- AWS EC2
- Git
- Nginx

## How to Use

1. terraform init
2. terraform plan
3. terraform apply

## Output
Displays public IP of the instance.
